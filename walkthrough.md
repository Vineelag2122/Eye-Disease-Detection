# Eye Disease Prediction App Walkthrough

## Overview
The Eye Disease Prediction app is now ready! It uses a Flask backend to serve a premium, modern frontend. The app loads your pre-trained MobileNetV2 model to predict eye diseases from uploaded images.

## Prerequisites
Ensure you have the required Python packages installed:

```bash
pip install -r requirements.txt
```

## Running the App
1. Open a terminal in the project directory.
2. Run the Flask app:

```bash
python app.py
```

3. Open your web browser and navigate to: `http://127.0.0.1:5000`

## Using the App
1. **Upload**: Click the upload area or drag and drop an eye image.
2. **Preview**: The image will be displayed in the preview area.
